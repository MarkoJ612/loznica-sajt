# City of Loznica Website

This is a **commercial-style website** created as a project for the Web Design course. It represents the city of Loznica, Serbia, showcasing its culture, tourism, sports, education, symbols, and more.

## Features

- Fully responsive design for desktop and mobile devices
- Multiple web pages including:

### Main Pages
- Home
- About the City
- News
- Contact
- Gallery

### City Sections
- Tourism
- Sports
- Culture
- Education
- City Symbols

### Information Pages
- Frequently Asked Questions (FAQ)
- Sitemap
- Press Page

## Technologies Used

- HTML5 & CSS3
- JavaScript (for dynamic date & time)
- Bootstrap 5 for layout and components
- Google Fonts
- Bootstrap Icons

## Live Demo

The website is hosted for free on Netlify: [https://gradloznica.netlify.app](https://gradloznica.netlify.app)

## Notes

- This project is primarily for educational purposes as part of a Web Design course.
- The hosted site should remain online as long as the Netlify account and repository are active.
- All code follows web design best practices covered in the course.
